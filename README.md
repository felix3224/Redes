## 🛠️ Tecnologias e Conceitos

- **Simuladores:** Cisco Packet Tracer, Wireshark
- **Protocolos de roteamento:** RIP (v1/v2), OSPF, BGP
- **Protocolos de rede/transporte/aplicação:** TCP, UDP, HTTP, DNS, ICMP
- **Fundamentos:** Modelo OSI e pilha TCP/IP, endereçamento IP e sub-redes (VLSM), encapsulamento de pacotes, análise de tráfego

git clone https://github.com/felix3224/Redes.git

## 💡 Principais Aprendizados

- Diferença prática entre protocolos de vetor de distância (RIP) e de estado de enlace (OSPF), e como cada um calcula e propaga rotas.
- Diagnóstico de problemas comuns: loops de roteamento, falhas de adjacência OSPF e inconsistências na tabela de roteamento.
- Análise de tráfego real com Wireshark, relacionando teoria (camadas OSI/TCP-IP) com o que efetivamente passa pela rede.
- Importância do planejamento de endereçamento IP e da documentação técnica no dia a dia de operação de redes.
